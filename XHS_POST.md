# 🔥 做了1个月的Chrome插件，今天终于上线了！

## 📌 先说重点

做了一个 **AI提示词管理插件**，支持：
- 10+ 提示词框架（BROKE、CO-STAR、CRISPE、RTF...）
- 🎨 **全新10维度图片生成器**（风格+质量+光照+构图+色调+材质+氛围+艺术流派+年代+负面词）
- 变量替换 {{变量名}} 语法
- 一键复制 / 自动填入到 ChatGPT/Claude/Gemini/DeepSeek
- 收藏夹 + 文件夹分类
- 本地存储，隐私安全

---

## 🎨 10维度图片生成器是什么？

之前每次写 Midjourney / Stable Diffusion 提示词都要想：
- 什么风格？写实还是动漫？
- 什么光线？逆光还是柔光？
- 什么色调？暖色还是赛博朋克？

现在 **点选就能组合** 👇

| 维度 | 可选标签 |
|------|----------|
| 风格 | 写实、科幻、奇幻、电影感、摄影、动漫... |
| 质量 | 4K、8K、超高清、无损、RAW... |
| 光照 | 自然光、逆光、霓虹灯、月光、体积光... |
| 构图 | 三分法、黄金螺旋、对称、留白... |
| 色调 | 暖色调、冷色调、黑白、赛博朋克... |
| 材质 | 金属、大理石、丝绸、毛绒、玻璃... |
| 氛围 | 宁静、悬疑、史诗、浪漫、梦幻... |
| 艺术流派 | 印象派、浮世绘、水墨、Blender渲染... |
| 年代 | 中世纪、文艺复兴、1920年代、近未来... |
| 负面词 | 低质量、模糊、噪点、畸形... |

**每个维度支持多选 + 自定义输入**，选完自动组合成完整提示词！

## 🔧 安装超简单

```bash
# 1. 克隆
git clone https://github.com/Ashtray8881/promptmaster.git

# 2. 安装
npm install && npm run build

# 3. Chrome 加载
# 打开 chrome://extensions/
# → 开启开发者模式
# → 加载已解压的扩展程序
# → 选择 dist/ 目录
```

**全程3分钟搞定！**

---

## 💡 怎么用的？

1. 点击插件图标打开侧边栏
2. 选择「图片生成器」模板
3. 10个维度逐个点选（可多选）
4. 底部实时预览组合结果
5. 复制 → 粘贴到 AI 绘画工具

**也可以自定义模板**，输入 `{{变量名}}` 作为占位符～
